### 1.2 值的类型
几种常见的类型：int（整数）, float(浮点数), str(字符串), bool(布尔值)  
int: 3  
float: 3.14  
str: 'hello world'  
bool: True  

### 1.3 计算
int: 整数可以+(加)、-(减)、*(乘)、//(整除)、%(取余)  
```python
1 + 1
2 - 1
3 * 5
6 // 2
5 // 3
6 % 2
5 % 3
```

float: 整数可以+(加)、-(减)、*(乘)、/(除)  
```python
1.0 + 1.0
2.0 - 1.0
3.0 * 5.0
6.0 / 2
5.0 / 3
```
**_注意：想看结果可以用print打印出来_**  

bool：布尔值可以or(或)、and（且）、not（非）  
```python
1 == 1 or 1 != 1
1 == 1 and 1 != 1
not 1 == 1
not 1 != 1
```


str: 字符串可以+（连接）  
```python
firstName = 'zhang'
lastName = 'san'
fullName = firstName + lastName
print(fullName)
```
  
类型转换：  
```python
piValue = float('3.14')
piName = str(3.14)
luckyNumber = int('12')
```
